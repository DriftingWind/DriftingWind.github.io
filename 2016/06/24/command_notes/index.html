android源码编译

在Android源码的下载、编译与导入到Android Studio 可能会出现下面的问题：
执行POSIXLY_CORRECT=1 sudo port install gmake libsdl git gnupg 出现 Port gmake not found

根据官方文档在安装了MacPorts之后，使用MacPorts安装依赖工具时，可能会遇到Port gmake not found的错误，在安装之前需要先执行下面的命令。

sudo port -d sync
然后就可以安装相关的工具了。

POSIXLY_CORRECT=1 sudo port install gmake libsdl git gnupg

如果安装始终失败，请直接写下载源码，下载完成后，再执行上面的命令试下
